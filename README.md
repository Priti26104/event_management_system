\# Event Management System



A web application for managing college events built with Spring MVC and deployed using Jenkins CI/CD.



\## Features

\- Create and manage events

\- Event registration

\- Participant tracking

\- Responsive web interface



\## Technology Stack

\- Java Spring MVC

\- JSP

\- Maven

\- Jenkins

\- Tomcat



\## Build \& Deploy

```bash

mvn clean package
